# PINF-6073_natural-computing
Natural computing course at UFES - 2019/2

## Requirements

- Pandas
- Matplotlib
- jupyter
- jupyter_contrib_nbextensions
- tqdm
- psutil
- memory_profiler
- line_profiler

Run

```
jupyter contrib nbextension install --user
```
